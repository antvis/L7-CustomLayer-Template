# L7-CustomLayer-Template

- 详细文档 https://github.com/antvis/L7-CustomLayer-Template/tree/master/docs/template%20api

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
